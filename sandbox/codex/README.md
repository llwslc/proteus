# Kit Candidate Theme Gallery

纯静态主题展馆，无框架、无依赖、无构建步骤。

- `index.html`：主题索引与预览切换器
- `themes/*.html`：15 个完全独立、可直接打开的主题页面

每页采用不同的产品场景和主交互：

| 主题 | 场景 | 独有主控件 |
| --- | --- | --- |
| SUMI | 墨池工作台 | 压感式水墨画布、笔架、砚台、落印 |
| ORMOLU | 镜厅席位簿 | 帷幕请柬、折扇配色、环形席位 |
| NOCTURNE | 夜花培育室 | 月时钟、花冠阶段、气味瓶、观察孔径 |
| PULP | 封面印刷床 | 嫌犯档案、砸印章、套色微调、翻页 |
| TOTEM | 织纹步进机 | 可编织网格、刻版纹样、染缸、踏板节拍 |
| KILN | 拉坯烧成室 | 可塑器形、转盘踏板、窑温弧表、测温锥 |
| CHROME | Liquid Player | CD 轨道环、凝胶均衡器、液柱音量 |
| ARCADE | 可玩迷宫柜 | 吃点迷宫、摇杆、投币口、生命槽 |
| BITMAP | 像素桌面系统 | 一位像素画板、拖拽窗口、菜单与保存框 |
| MOCHI | 泡芙装配台 | 挤压、咬缺、蒸汽阀、装盒柱塞 |
| SCRIBBLE | 手账作战本 | 可拖便利贴、发条计时器、胶带与贴纸盒 |
| AXON | 爆炸装配图 | 轴测分层、爆炸间距、剖面线、尺寸标注 |
| CORONA | 日食观测台 | 食相轨道、谱线滤镜、贝利珠、曝光鼓 |
| QALAM | 书法装帧室 | 笔锋角度、诗律节拍、八角花饰、收笔花押 |
| GAZETTE | 晚报排版台 | 活头版、栏数排版、故事优先级、对版十字 |

任意静态文件服务器均可运行本目录。
